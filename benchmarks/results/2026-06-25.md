## Methodology

See `evals/README.md` and `shared/BENCHMARK_PROMPTS.md`.  Three arms per (dialect, prompt) pair:
- **A:**  `claude --print --bare --model <m>`  (no system prompt).
- **B:**  same with `--system-prompt "Answer concisely."`.
- **C:**  same with `--system-prompt "$(cat <dialect-skill.md>)"`.

Each row's metric is the median across the prompt set, not the mean.  Single run per (prompt, arm) at default temperature.  Caveats per the spec disclaimers:  word-token approximation, no statistical significance testing, no cross-model verification.
